## Tela de Seleção de Procedimento

- Lista organizada de procedimentos médicos disponíveis
- Campo de busca para localização rápida de procedimentos
- Filtros por categoria de procedimento (coleta, análise, etc.)
- Lista de procedimentos recentes ou frequentes
- Botão claro "Selecionar" para avançar com o procedimento escolhido
- Botão "Voltar" para retornar à tela inicial
